SETUP INSTRUCTIONS:
-------------------

1. CREATE IMAGES FOLDER
   - Create a new folder called "images" in the same location as index.html
   
2. ADD YOUR IMAGES
   Place your 6 images in the images folder with these exact names:
   
   ✓ hero-image.jpg    - Your Keto Life Café kitchen/landing page photo
   ✓ burger.jpg        - Lettuce wrap burger image
   ✓ pizza.jpg         - Cauliflower crust pizza image
   ✓ curry.jpg         - Coconut chicken curry image
   ✓ cauliflower.jpg   - Turmeric roasted cauliflower image
   ✓ pasta.jpg         - Zucchini pesto noodles image
   
   NOTE: You can use .jpg, .jpeg, or .png formats. If you use different 
   names or formats, just update the image paths in index.html

3. OPEN THE WEBSITE
   - Double-click index.html to view in your browser
   - Or use a local server for testing

4. UPLOAD TO WEB SERVER
   Upload ALL files to your web hosting:
   - index.html
   - error.html
   - style.css
   - images/ folder (with all images inside)


CUSTOMIZATION:
--------------

COLORS:
- Primary green: #16a34a (change in style.css)
- Search for "#16a34a" in style.css to update all green colors

TEXT CONTENT:
- Open index.html in a text editor
- Edit any text between HTML tags
- Save and refresh browser

IMAGES:
- Replace images in the images/ folder
- Keep the same filenames OR update paths in index.html

STYLING:
- All styles are in style.css
- Edit fonts, sizes, spacing, colors as needed


CONTACT FORM:
-------------
